May 01, 2020点击

这一节我将带着大家来初始化一个前端项目，创建前端项目我们会使用到vue cli 3这个工具，是一个脚手架工具，所谓脚手架就是一个项目初始化的结构，vue cli 3帮助我们规范了项目初期的目录结构，结构配置等等，然后我们可以多把事件花在逻辑层上，减少了繁琐的添加各种配置，但是也不排除我们在项目开发过程中会修改一些配置来满足我们的项目需求。

初始化前端项目

首页我们需要安装vue cli来初始化我们的前端项目框架（npm下载很慢大家可以安装一下淘宝镜像cnpm）：

npm install -g @vue/cli

然后我们使用vue/cli创建一个项目： wecircle

vue create wecircle


选择Manually select features,上下键选择，回车确认，空格选中，表示我们不采用默认的模板，而是根据自己的情况选择需要安装的模块，例如vue-router，ESLint等等。
